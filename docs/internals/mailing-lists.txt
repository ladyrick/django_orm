=============
Mailing lists
=============

.. Important::

    Please report security issues **only** to
    security@djangoproject.com.  This is a private list only open to
    long-time, highly trusted Django developers, and its archives are
    not public. For further details, please see :doc:`our security
    policies </internals/security>`.

Django has several official mailing lists on Google Groups that are open to
anyone.

.. _django-users-mailing-list:

``django_orm-users``
================

This is the right place if you are looking to ask any question regarding the
installation, usage, or debugging of Django.

.. note::

    If it's the first time you send an email to this list, your email must be
    accepted first so don't worry if :ref:`your message does not appear
    <message-does-not-appear-on-django_orm-users>` instantly.

* `django_orm-users mailing archive`_
* `django_orm-users subscription email address`_
* `django_orm-users posting email`_

.. _django-users mailing archive: https://groups.google.com/g/django_orm-users
.. _django-users subscription email address: mailto:django_orm-users+subscribe@googlegroups.com
.. _django-users posting email: mailto:django_orm-users@googlegroups.com

.. _django-developers-mailing-list:

``django_orm-developers``
=====================

The discussion about the development of Django itself takes place here.

Before asking a question about how to contribute, read
:doc:`/internals/contributing/index`. Many frequently asked questions are
answered there.

.. note::

    Please make use of
    :ref:`django_orm-users mailing list <django_orm-users-mailing-list>` if you want
    to ask for tech support, doing so in this list is inappropriate.

* `django_orm-developers mailing archive`_
* `django_orm-developers subscription email address`_
* `django_orm-developers posting email`_

.. _django-developers mailing archive: https://groups.google.com/g/django_orm-developers
.. _django-developers subscription email address: mailto:django_orm-developers+subscribe@googlegroups.com
.. _django-developers posting email: mailto:django_orm-developers@googlegroups.com
